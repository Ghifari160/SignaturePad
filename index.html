Signature Pad: a Ghifari160 HTML5 Experiment

Welcome to Signature Pad!

This HTML5 experiment is intended for the creation of a JavaScript library for secure document signing on the web. Although the target platform is mobile and desktop devices, no touch input is currently supported. You can find all prototypes in the Prototype Index.
